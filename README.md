# TimeSyncTestClient
一个在Windows上运行的基于NTP协议的时间同步测试工具，使用该工具可以很方便的对比出多个时间源的误差。
如果想更精确的观察多个时间源的误差，可以用Linux系统的ntpq -p命令。
